# Cycle de développement

Dans quel ordre devrait se passer un développement
- écriture de code
- analyse
- édition des liens
- compilation
- éxecution

<details>
<summary>Answer</summary>
- analyse
- écriture de code
- compilation
- édition des liens
- éxecution
</details>

<details>
<summary>Solution</summary>

1. analyse
2. écriture de code
3. compilation
4. édition des liens
5. éxecution

**NB** : la compilation et l'édition est faite en séquence souvent de manière *transparente* 

</details>
